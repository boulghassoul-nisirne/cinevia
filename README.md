 Cinevia - Application de découverte de films
 Description

Cinevia est une application web moderne dédiée à la découverte de films. Elle permet aux utilisateurs de parcourir une collection de films, rechercher facilement, et consulter les détails de chaque film.

 Fonctionnalités
 Recherche de films en temps réel
 Affichage des films sous forme de cartes
 Page de détails pour chaque film
 Ajout aux favoris
 Interface moderne et responsive
 Pages
index.html : Page principale (liste des films)
details.html : Détails du film
favoris.html : Liste des films favoris
 Technologies utilisées
HTML5
CSS3 
JavaScript 
LocalStorage pour sauvegarder les favoris
 Installation et utilisation
Ouvrez index.html dans votre navigateur
Parcourez les films disponibles
Utilisez la barre de recherche pour filtrer
Cliquez sur "Voir détails" pour plus d'informations
Ajoutez des films aux favoris 
 Structure des données

Chaque film contient :

Titre
Image (URL)
Description
Genre
ID unique
